Proton Cloudflare Domains Web Key Directory
===========================================
This sets up a [Web Key Directory](https://wiki.gnupg.org/WKD) for your [Proton Mail](https://proton.me/mail) email address that is using
a custom domain, where the domain is hosted by Cloudflare.

Please [read the wiki](https://github.com/finallychristine/proton-cloudflare-wkd/wiki)
for complete instructions and setup.
